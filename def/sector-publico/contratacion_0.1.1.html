Appealc back to ToC or Class ToC

IRI: http://contsem.unizar.es#Appeal

Cuando se interponga un recurso sobre alguna de las resoluciones del proceso de licitación se utilizará la propiedad contsem:appeal con rango en la clase contsem:Appeal.

is in domain of
Resultado del recurso dp, Tipo de recurso op

Comité de contrataciónc back to ToC or Class ToC

IRI: http://contsem.unizar.es#TenderCommittee

Esta clase puede representar una de las siguientes tres entidades, dependiendo de la propiedad utilizada:

- Mesa de contratación (contsem:tenderCommittee)

- Comité de expertos (contsem:expertCommittee)

- Organismo técnico (contsem:technicalCommittee)

is in domain of
Fecha de nombramiento dp

Contract extinctionc back to ToC or Class ToC

IRI: http://contsem.unizar.es#ContractExtinction

Existe una serie de circunstancias que derivan en la extinctión del contrato. Para describir estas situaciones se hace uso de la clase contsem:ContractExtinction. El tipo de extinción vendrá determinado por la propiedad que se utilice para enlazar a esta clase desde el contrato. Existen tres propiedades para ello:

- Resolución (contsem:contractResolution)

- Renuncia (contsem:contractWaive)

- Abandono (contsem:contractAbandonment)

is in range of
Contract extinction op

Contract modificationc back to ToC or Class ToC

IRI: http://contsem.unizar.es#ContractModification

Durante un contrato se puede dar también la necesidad de modificarlo una vez iniciada su ejecución, para plasmar este tipo de casos en la ontología CONTSEM se debe utilizar la clase contsem:ContractModification (rango de la propiedad contsem:contractModification).

is in range of
Modificación de contrato op

IRI: http://contsem.unizar.es#BundlePriceSpecification

Clase que representa el precio de conjuntos de objetos o un paquete determinado.
